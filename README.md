# My Shopping App

Angular 購物車練習專案 — 包含書籍列表、購物車明細、登入/登出等功能。

## 環境需求

- [Node.js](https://nodejs.org/) v18 以上
- npm（隨 Node.js 一起安裝）

## 安裝依賴

```bash
npm install
```

## 啟動開發伺服器

```bash
npx ng serve
```

啟動後開啟瀏覽器前往 `http://localhost:4200/`，修改程式碼後頁面會自動重新載入。

## 建置專案

```bash
npx ng build
```

產出的檔案會在 `dist/` 目錄中。
